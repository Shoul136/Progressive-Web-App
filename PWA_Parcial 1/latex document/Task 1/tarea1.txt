Que es una pwa?
ES una aplicacion web que combina caracteristicas de un navegador y puede usar funcionalidades movil, estas se pueden acceder desde cualquier navegador y pueden ser descargadas como si fuera una app nativa
https://immune.institute/blog/que-es-pwa/

Caracteristicas de una pwa?
Actualizada: Las pwa siempre mostraran la ultima version al usuario, con el empleo de automatizaciones automaticas, de manera constante sin necesidad de descargarlas.
Segura: AL tener caracteristicas web usan el protocolo https, lo cual le da un nivel de seguridad adicional a este tipo de aplicaciones ya que se usan tecnologias como TLS
Rapidas: Tienen la velocidad de carga de un navegador, optimizacion, ademas que guardan cosas en la memoria cache acelerando el proceso de busqueda.
Offline: Pueden funcionar sin internet, de forma que puedes cargar paginas para los usuarios y hacerle saber que aun no hay alguna conexion mejorando la experiencia del usuario.
